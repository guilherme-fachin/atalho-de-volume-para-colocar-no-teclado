# atalho-de-volume-para-colocar-no-teclado
é um código script AutoHotckey que tem como obj. facilitar o controle sobre o volume, que enves de precisar aumentar o volume usando o mouse, agora com script permite que o controle do volume seja acessado pelas teclas: "Pause Break","ScrLk", "Insert". Sendo que "Pause Break" = diminuir o volume,"ScrLk" = bloquear/desbloquear e "Insert" = aumentar
